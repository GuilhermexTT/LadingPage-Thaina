# Dra. Thainá Carvalho - Site Oficial

Site oficial e landing page da Dra. Thainá Carvalho, focada em Harmonização Facial e Corporal, desenvolvido com alta performance e design premium.

## 🚀 Tecnologias

* **Next.js 14** e **React 18**
* **Tailwind CSS** para estilização
* **Sanity CMS** para gerenciamento de conteúdo dinâmico

## 💻 Instalação e Execução

Para rodar o projeto em sua máquina local, siga as instruções:

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse no navegador:
   * **Site:** [http://localhost:3000](http://localhost:3000)
   * **Painel Administrativo (CMS):** [http://localhost:3000/admin](http://localhost:3000/admin)

## 🎨 Sobre o Projeto

O projeto foi construído priorizando:
- **Performance e SEO:** Renderização otimizada para carregamento rápido.
- **Responsividade:** Experiência fluida em dispositivos móveis e desktops.
- **Integração Dinâmica:** Seções de procedimentos, depoimentos e fotos de resultados são facilmente atualizáveis pelo painel integrado.
